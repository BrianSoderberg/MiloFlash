MILO Introduction

It's a smart (micro controller powered) platform that has a couple unique features:

  1. It's Intelligent — for Boss devices, like the Katana Go, MILO syncs the device, meaning it reads the entire device and saves all the information. So, after syncing, when you start MILO it will read its Katana Go saved configuration. This allows MILO to present you with all the presets, along with their titles. You can navigate through the 30 presets, and when you select one, it takes you to the effects page associated with that preset with the on/off states shown. From the effects page you can toggle effects on and off, as well as the pedal effects.
  2. Create your own effects (Stomps) and save them — You can go to the effect edit page and set the parameters of any effect any way you want. From any preset, you can turn your own effect creations on and off on Stomps page 1 and 2 (16 extra effects).
  3. Create custom expression pedal parameter controls — from the effect editor page, you can set nearly any parameter (100's to choose from) to any of the 4 expression pedal jacks. For example one Stomp on/off pedal to enable the Amp volume and gain to be set to expression pedals 3 and 4. Super useful. You could also assign the Solo level to expression jack 0, to adjust during a solo. The options with this are limitless.

All the Stomps (effects, expression pedals and snaps) are saved to 16 footpedal slots, to be used during any preset.
